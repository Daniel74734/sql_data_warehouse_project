# Data Warehouse Project

Welcome to the Data Warehouse repository. This project focuses on building a modern data warehouse using SQL Server, covering ETL processes, data modeling, and data analytics.

---

## Project Requirements

### Objective

Develop a modern data warehouse using SQL Server that brings data together from different sources into one place, making it easier to analyze and report on.

---

### Specifications

**Data Sources**
- Data is pulled from two source systems: ERP and CRM
- The data comes in CSV file format

**Data Quality**
- Only clean, accurate, and complete data is loaded into the warehouse
- Any duplicate or missing records are handled before loading

**Integration**
- Data from all sources is combined into a single, unified data model
- This makes it easy to query and analyze everything from one place

**Scope**
- This project covers data from the year 2020 to 2024
- It focuses on sales and customer data only

**Documentation**
- All tables and processes are clearly named and easy to follow
- This README serves as the main guide for understanding the project

---

## BI: Analytics & Reporting

### Objective

Develop SQL-based insights from the data warehouse to help answer business questions and support decision-making.

- Total sales by product, region, and time period
- Customer behavior and segmentation
- Monthly and yearly performance trends
- Key business metrics presented as simple SQL queries and views
